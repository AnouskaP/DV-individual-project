7CCSMSDV Data Visualisation Project 2024

Introduction

These visualisations serve to answer Research Question 3 of part 3 of the Data Visualisation exam:
Analyse performances of stock prices over time. Are there any detectable trends?
There are 3 visualizations which can be accessed through the top navigation bar:

  • Line Chart
  • A comparison line chart
  • Candlestick chart

Calculation for Top Performing Companies

  • The calculation focuses on the opening and closing prices of the stock for the first and last days of the specified period.
  • Percentage Change Calculation: The percentage change is calculated using the formula:
    \[ \text{Percentage Change} = \left( \frac{\text{Closing Price} - \text{Opening Price}}{\text{Opening Price}} \right) \times 100 \]
  • Aggregate Calculation: For each company, the percentage changes calculated for individual periods (e.g., months) are aggregated to determine the overall performance over the specified time frame. I used Python to perform these calculations.
